# add-shallow

Generate shallow render react test boilerplate

## Install

```sh
$ npm install -g add-shallow
```

## Usage

```sh
# Generate index.test.js in current directory
$ add-shallow

# Generate index.test.js in custom path
$ add-shallow --path my/component/directory
```
